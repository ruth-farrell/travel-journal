# Travel Journal

Travel Journal is a personal travel web app for browsing destinations, filtering trips, and exploring travel memories in a visual way.

## What It Does

- Displays destination cards with trip details, country flags, dates, descriptions, and tags.
- Supports filtering by year, month, country, and tag.
- Includes search and sort controls to quickly find places.
- Offers an interactive world map for country-based filtering.
- Shows extra destination photo slides in modal galleries.

## Tech Stack

### Frontend

- React 17 + TypeScript
- Create React App (`react-scripts`)
- SASS for styling
- React Router
- React Icons and Flag Icons
- Testing Library + Vitest

### Backend

- Node.js + Express
- MongoDB + Mongoose
- `dotenv` for environment configuration

## Local Development

Install dependencies:

```bash
npm install
```

Run the frontend:

```bash
npm start
```

Run the backend API:

```bash
npm run server
```

Build for production:

```bash
npm run build
```
